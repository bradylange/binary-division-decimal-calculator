## `src/divide.c`
This program takes two integers and divides them using the binary division algorithm.
It will output the quotient and remainder. It also cannot use / or %, only bitwise
operators. 

Input: `a.out 10 5`

Output: 

`Quotient: 10 5`

`Remainder: 0`
